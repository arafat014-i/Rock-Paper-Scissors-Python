# 🎮 Rock Paper Scissors Game

A simple Python game where the user plays **Rock–Paper–Scissors** against the computer.

The computer randomly selects a move and the program determines the winner based on classic game rules.

---

## ✨ Features

* Play against the computer
* Random computer choice using Python
* Simple command-line interface (CLI)
* Beginner-friendly Python project

## ⚙️ Requirements

* Python 3.x installed on your computer

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/arafat014-i/rock-paper-scissors-python.git
```

Go to the project directory:

```bash
cd rock-paper-scissors-python
```

Run the game:

```bash
python rock_paper_scissors.py
```
