# 🎮 Rock Paper Scissors Game (Python)

A simple **Rock–Paper–Scissors game** written in Python where the user plays against the computer.

The computer randomly selects a move and the program determines the winner based on the classic game rules.

---

## 📌 Features

* Play Rock–Paper–Scissors against the computer
* Random computer move using Python's `random` module
* Simple command-line interface
* Beginner-friendly Python project

---

## 🧠 Game Rules

* **Rock beats Scissor**
* **Scissor beats Paper**
* **Paper beats Rock**
* If both players choose the same move → **Tie**

## ▶️ How to Run the Game

Clone the repository:

```bash
git clone https://github.com/arafat014-i/rock-paper-scissors-python.git
```

Go to the project folder:

```bash
cd rock-paper-scissors-python
```

Run the program:

```bash
python rock_paper_scissors.py
```
